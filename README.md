LED와 조도 센서를 활용한 별의 등급 모형 실험
본 프로젝트는 별의 겉보기등급과 절대등급이 거리에 따라 어떻게 달라지는지 알아보기 위해 진행되었다. 초기에는 조도 센서, 공기질 센서, HuskyLens, LED 바를 모두 연결한 복합 장치를 구상했으나, 탐구 목적을 고려하여 조도 센서와 WS2813 LED 바만 활용하는 방식으로 실험을 단순화하였다.

WS2813 LED 바는 휴대전화의 웹 대시보드를 통해 LED의 밝기와 점등 개수를 조절할 수 있도록 제작하였다. 조도 센서는 LED에서 방출되는 빛의 세기를 측정하고, 설정한 범위에 따라 등급을 결정하도록 코딩하였다. 두 장치는 서로 다른 컴퓨터에서 각각 제어하였다.

초기 실험에서는 상대등급과 LED 개수가 일정한 값으로만 출력되는 문제가 발생했다. 원인을 분석한 결과, 등급을 구분하는 조도 범위를 지나치게 넓게 설정했기 때문임을 확인하였다. 이에 사전 측정 데이터를 바탕으로 범위를 재설정하고 실험 장소를 변경하여 총 네 가지 조건에서 실험을 진행하였다.

측정 범위가 충분히 세분되지 않아 정밀한 데이터를 얻는 데에는 한계가 있었지만, 조건별 대략적인 값을 산출하고 서로 비교할 수 있었다. 마지막으로 측정 결과를 포그슨 공식과 거리 지수 공식에 적용하여 별의 절대등급을 계산하였다.

핵심 탐구 과정
장치 구상 → 기능 단순화 → 웹 대시보드 제작 → 조도 측정 → 오류 원인 분석 및 범위 재설정 → 네 가지 조건 비교 → 절대등급 계산

기대 효과
이 실험을 통해 별의 밝기가 거리와 관측 조건에 따라 다르게 나타나는 원리를 LED와 조도 센서로 모형화하고, 겉보기등급과 절대등급의 관계를 실제 측정값과 수식을 통해 탐구할 수 있었다.
