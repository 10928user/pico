처음에는 조도센서, 공기질 센서, 허스키렌즈, LED 바 등을 모두 연결해 유기적으로 작용하는 실험 장치를 제작하려고 시도했다. 하지만 별의 절대등급과 겉보기등급의 주 원인은 별의 거리가 크기 때문에 LED 바와 조도센서를 제외한 다른 센서들은 사용하지 않았다. 처음에는 조도센서와 LED 등을 이용해서 조도센서가 LED등에 의해 영향을 받는지 테스트 했고 이후 구상한 실험이 실현 가능한지 검토했다.
이후,  기능을 단순화해서 HuskyLens를 빼고, WS2813 LED 바를 휴대전화로 제어하는 웹 대시보드를 만드는 방향으로 바꾸었다.
LED 등과 조도 센서 코딩은 서로 다른 컴퓨터에서 진행했다. LED 등의 경우에는 밝기와 킬 수 있는 LED 개수를 대시보드 상에서 조정할 수 있도록 했다. 조도 센서의 경우, 등급의 범위를 입력해 결정했다. 
처음에 실행했을 때 상대등급과 LED 개수 값이 고정되어 출력되었는데 그 이유가 등급의 범위를 너무 넓게 잡았기 때문임을 알게 되었다. 그래서 앞서 추출한 데이터를 기반으로 하여 범위를 재조정하고 장소를 옮겨서 총 네 가지의 경우를 고려해 실험했다. 범위를 많이 만들지 못해서 데이터를 세부화시키지는 못했지만 대략적인 값은 산출되어서 이를 서로 비교해볼 수 있었다. 이후 AI를 이용해서 포그슨 공식과 절대등급 공식에 대입하여 절대등급을 구하였다.
